# Локальная копия silagranita69.ru

Сайт собран как многостраничный:

- `index.html` — главная страница;
- `kuhni/index.html` — кухонные столешницы;
- `pamyatniki/index.html` — рекламная страница памятников;
- `assets/css/design-system.css` — общая дизайн-система;
- `assets/js/site.js` — мобильное меню, формы и карусели;
- `assets/images/kuhni/` — изображения страницы кухонь.

На главной находятся каталог продукции, производственные мощности, портфолио, отзывы, форма расчёта, контакты и карта производства.

Для просмотра запустите папку `site-copy` через расширение VS Code/Cursor **Live Server**:

1. Откройте `index.html`.
2. Нажмите **Open with Live Server**.

Это статическая копия опубликованной страницы Tilda. Внешние ссылки оставлены рабочими, но отправка форм, аналитика и другие серверные интеграции Tilda могут продолжать зависеть от исходного сайта.
